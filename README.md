# Card Draw Probability Calculator 🃏

**เครื่องคำนวณความน่าจะเป็นในการจั่วการ์ด** — เครื่องมือสำหรับวิเคราะห์โอกาสในการจั่วการ์ดที่ต้องการในเกมการ์ดต่างๆ (TCG/Board Games) รองรับการคำนวณด้วย **Multivariate Hypergeometric Distribution**

---

## 🚀 ฟีเจอร์หลัก
* **คำนวณแม่นยำ:** ใช้ Multivariate Hypergeometric Distribution รองรับการ์ดหลายชนิดพร้อมกัน
* **เงื่อนไขยืดหยุ่น:** รองรับตัวดำเนินการเปรียบเทียบ (`<`, `≤`, `=`, `≥`, `>`) สำหรับแต่ละชนิดการ์ด
* **Data Visualization:** สร้างกราฟวิเคราะห์ความน่าจะเป็นเมื่อเปลี่ยนตัวแปรต่างๆ
* **UX/UI:** รองรับ Dark/Light Mode และรองรับสองภาษา (ไทย/อังกฤษ)
* **ใช้งานง่าย:** ไม่ต้องติดตั้ง (Client-side) เพียงเปิด `index.html` ผ่าน Browser

---

## 🛠 วิธีการใช้งาน
1. **ตั้งค่ากองการ์ด:** ระบุจำนวนการ์ดทั้งหมดในกอง (`N`) และจำนวนการ์ดที่จะจั่ว (`n`)
2. **เพิ่มชนิดการ์ด:** กดปุ่ม **"+ เพิ่มชนิด"** เพื่อระบุชื่อการ์ด, จำนวนที่มีในกอง (`Kᵢ`) และกำหนดเงื่อนไขการจั่ว
3. **คำนวณ:** กด **"คำนวณความน่าจะเป็น"** ระบบจะแสดงผลเป็นเปอร์เซ็นต์, ทศนิยม, และ Odds Ratio
4. **สร้างกราฟ:** เลือกตัวแปรที่ต้องการ Sweep ช่วงค่า แล้วกด **"สร้างกราฟ"** เพื่อดูแนวโน้มความน่าจะเป็น

## ⚖️ ตัวดำเนินการเปรียบเทียบ

| ตัวดำเนินการ | ความหมาย | ตัวอย่างการใช้งาน |
| :---: | :--- | :--- |
| `<` | น้อยกว่า N ใบ | `< 2` (ได้ 0 หรือ 1 ใบ) |
| `≤` | ไม่เกิน N ใบ | `≤ 2` (ได้ 0, 1, หรือ 2 ใบ) |
| `=` | เท่ากับ N ใบพอดี | `= 2` (ได้ 2 ใบเท่านั้น) |
| `≥` | อย่างน้อย N ใบ | `≥ 2` (ได้ 2 ใบขึ้นไป) |
| `>` | มากกว่า N ใบ | `> 1` (ได้ 2 ใบขึ้นไป) |

> **หมายเหตุ:** ระบบจะคำนวณจากเงื่อนไขทุกชนิดที่ต้องเป็นจริง **พร้อมกัน** เท่านั้น

## 📐 หลักคณิตศาสตร์
ใช้สูตรการคำนวณความน่าจะเป็นแบบหลายตัวแปร:

$$P = \frac{\sum \prod_{i} \binom{K_i}{x_i} \cdot \binom{R}{n - \sum x_i}}{\binom{N}{n}}$$

* **N**: จำนวนการ์ดทั้งหมดในกอง
* **n**: จำนวนการ์ดที่จั่ว
* **Kᵢ**: จำนวนการ์ดชนิดที่ i ในกอง
* **xᵢ**: จำนวนการ์ดชนิดที่ i ที่จั่วได้
* **R**: จำนวนการ์ดที่เหลือที่ไม่ได้ระบุชนิด
* *เทคนิค:* ใช้ **log-space arithmetic** เพื่อป้องกันค่า Overflow ในกรณีที่ Deck มีขนาดใหญ่
